# FIAP_Tech-Challenge_Fase3
Este projeto tem como objetivo analisar os dados da PNAD COVID-19, explorando o comportamento da população brasileira durante a pandemia, com foco em indicadores de saúde, sintomas, procura por atendimento médico e impactos socioeconômicos.

O pipeline de dados foi desenvolvido utilizando AWS S3, AWS Glue, Amazon Athena e PySpark, contemplando as etapas de ingestão, catalogação, processamento e agregação dos dados. Os resultados foram organizados em camadas analíticas e visualizados por meio de dashboards no Power BI, permitindo a extração de insights relevantes.
